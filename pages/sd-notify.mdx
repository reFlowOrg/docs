Next Generation Notification System

## Notify

Simple Notification

<Tabs items={['Client', 'Server']}>
  <Tab>
    ```lua copy
    exports['sd-notify']:Notify(title, message, time, type, position, hideprogressbar, sound)
    ```
  </Tab>
  <Tab>
    ```lua copy
    TriggerClientEvent('sd-notify:Notify', source, title, message, time, type, position, hideprogressbar, sound)
    ```
  </Tab>
</Tabs>

- title: `string`
- message: `string`
- time: `number`
- type: [Types](/sd-notify/types)
- position?: [Position](/sd-notify/positions)
- hideprogressbar?: `boolean`
- sound?: `boolean`

returns: `string`
