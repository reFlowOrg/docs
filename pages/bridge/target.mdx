# Target

## DisableTarget

Disables target

```lua
Target.DisableTarget(state)
```

- state : `boolean`

## AddGlobalObject

Create Global Object Target

```lua
Target.AddGlobalObject(options)
```

- options : [Options[]](/bridge/target/options)

## RemoveGlobalObject

Remove Global Object Target

```lua
Target.RemoveGlobalObject(labels)
```

- labels : `string | string[]`
