# Target BoxZone

Creates Targetable BoxZone.

```lua
Target.AddBoxZone(data)
```
- data: `table`
    - debug?: `boolean`
    - name: `string`
    - coords: `vector3`
    - size: `vector3`
    - rotation: `number`
    - options: [Options[]](/bridge/target/options)

returns: `number | string`

```lua
Target.AddBoxZone({
    name = 'MyBoxZone',
    coords = vector3(0, 0, 0),
    size = vector3(1, 1, 1),
    rotation = 0,
    debug = true,
    options = {}
})
```
