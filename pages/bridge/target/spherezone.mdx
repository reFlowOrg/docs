# Target SphereZone

Creates Targetable SphereZone.

```lua
Target.AddSphereZone(data)
```
- data: `table`
    - debug?: `boolean`
    - name: `string`
    - coords: `vector3`
    - radius: `number`
    - options: [Options[]](/bridge/target/options)

returns: `number | string`

```lua
Target.AddSphereZone({
    name = 'MySphereZone',
    coords = vector3(0, 0, 0),
    radius = 1,
    debug = true,
    options = {}
})
```
